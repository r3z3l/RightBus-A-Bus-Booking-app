# Bus Booking System

## Installation

1. ADD .env in bus-booking server
```
DATABASE_URL=YOUR_DATABASE_URI
```

2.  Installing packages for Backend 
```
    cd bus-booking-server
    npm install
    npm run start
```

3. Installing packages for frontend
```
    cd rightbus-frontend
    npm install
    npm run dev
```
